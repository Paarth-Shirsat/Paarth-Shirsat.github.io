Projects

Smart-Watch

Github: https://github.com/Paarth-Shirsat/Smart-watch
Desc: An industry-grade smartwatch from scratch. This project focuses on developing a custom, compact smartwatch on a dedicated PCB that integrates both hardware and Zephyr RTOS-based firmware components. The overall goal is to build a complete embedded wearable system by combining hardware design, firmware development, and user interface implementation.

Handheld Gaming Console

Github: https://github.com/Paarth-Shirsat/Handheld-Gaming-Console
Desc: The Handheld Gaming Console is a custom made Printed Circuit Board which utilizes STM32F407VET6 as its MCU controlling all of the peripherals. The firmware involves setting up all the components and utilizing them to run simple games with the help of controlling hardware such as buttons, joystick and IMU. The firmware for this project is a bare-metal implementation built on HAL. This project is designed to provide hands-on experience and a deep understanding of the embedded development workflow.

12 Hour Clock

Github: https://github.com/SarvaarthN/12-Hr-Clock
Desc: This project involves the design and implementation of a non-microcontroller-based digital clock using crystal oscillators, decade counters, and seven-segment display driver ICs.
